# Chrome Web Store 審査者向けメモ

## 拡張機能の目的

GemMurmurは、閲覧中の一般的なWebページに、ページ内容に関連した短いライブコメントを重ねて表示する拡張機能です。コメント生成は、ダウンロード済みのGemmaモデルを使って端末内で行います。

## 確認手順

1. WebGPUに対応したデスクトップ版Chromeで拡張機能を開きます。
2. ポップアップの「モデルを準備する」を選択します。初回のみ約2.0GBのGemmaモデルをHugging Faceから取得します。
3. モデルの準備が完了し、ポップアップが `Ready` になったことを確認します。
4. ログインが不要な一般Webページを開くか再読み込みします。
5. ページ上に、右から左へ流れる短いコメントが表示されることを確認します。スクロールや連続クリックでもコメントが追加されます。

## プライバシー上の挙動

- ページのタイトル・見出し・表示中の本文、URL・ドメインは、コメント生成と停止判定のため端末内で扱います。
- これらのページデータをGemMurmurのサーバー、外部AI API、広告事業者、その他の第三者へ送信しません。
- 初回のGemmaモデル取得ではHugging Faceに接続しますが、ページ本文や閲覧URLは送信しません。
- ログイン、決済、メールなどのプライベートページでは、コメントを自動停止します。

## 必要な権限

- `storage`: 表示・言語・停止・モデル状態の設定を端末内に保持します。
- `tabs`: 現在のページだけを識別し、ページ／サイト単位の停止設定に使います。
- `alarms`: 長時間利用時のコメント密度の管理に使います。
- `offscreen`: ポップアップを閉じた後も端末内のGemmaモデルを読み込みます。
- 一般ページへのアクセス: ページ上にコメントを表示し、端末内でページ内容を処理します。
- Hugging Face関連ドメイン: 初回のGemmaモデル取得だけに使います。

## 補足

モバイル版Chromeでは拡張機能を実行できません。デスクトップ版ChromeまたはChromium版Microsoft Edgeを対象としています。
