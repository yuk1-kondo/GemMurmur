GemMurmur プライバシーポリシー

最終更新日: 2026-07-22

GemMurmur（以下「本拡張機能」）は、閲覧中の一般 Web ページに短いライブコメントを表示する Chrome 拡張機能です。本ポリシーでは、本拡張機能が扱う情報とその目的を説明します。

1. 扱う情報

本拡張機能は、現在のページの URL・ドメイン・タイトル・説明・見出し、表示中または周辺のページ本文と画像の代替テキスト、ページ種別と言語の推定結果、スクロール・クリック・ページ遷移・タブ復帰・放置などの閲覧操作、ON/OFF・コメント言語・停止設定・モデル状態を端末内で処理します。

ログイン、決済、メール、医療・金融に関係する URL やフォームなど、プライベートな可能性があるページを検知した場合、本拡張機能はコメント生成と表示を停止するよう設計されています。

2. 利用目的

上記の情報は、ページに関連した短いコメントを生成・表示し、利用者が選んだ設定を維持する目的にのみ使用します。コメント生成は利用者の端末上で WebGPU とローカル Gemma モデルを用いて行います。

3. 外部送信、共有、販売

ページ内容、URL、閲覧操作、生成コメント、設定を、開発者のサーバー、広告事業者、分析事業者、またはその他の第三者へ送信、共有、販売しません。これらの情報を広告配信、ユーザープロファイリング、信用判断に利用しません。

利用者が「モデル読み込み」を選んだ場合に限り、ローカル実行用の Gemma モデルファイルを Hugging Face のドメインから HTTPS で取得します。この通信には、ページ内容や閲覧操作は含まれません。

4. 保存と削除

設定とモデルの状態は Chrome のローカル拡張機能ストレージに保存されます。ダウンロードしたモデルは端末内のブラウザキャッシュに保存され、ポップアップの「削除」から削除できます。拡張機能をアンインストールすると、Chrome が拡張機能に紐づくローカルデータを削除します。

5. セキュリティ

モデル取得は HTTPS を使用します。本拡張機能は機能に必要な最小限の Chrome 権限を使用するよう設計されています。プライベートなページでは本拡張機能を無効にすることを推奨します。

6. ポリシーの変更

本ポリシーまたはデータの取扱いを変更する場合は、このページと Chrome ウェブストアの開示を更新します。

7. お問い合わせ

お問い合わせは GemMurmur の GitHub Issues からお願いします。Issue にページ本文、URL、認証情報などの機微な情報を投稿しないでください。
